## 進め方

| フェーズ | 内容 | 主なアウトプット |
| --- | --- | --- |
| 1. 情報整理 | 公式情報、試験範囲、参考資料を集める | 参考リンク、試験範囲メモ |
| 2. 学習 | テーマ別に学習し、試験ポイントを整理する | 学習ログ、弱点リスト |
| 3. 実践 | 業務で使える小さな活用例を試す | 実務活用ログ、プロンプト例 |
| 4. 成果物化 | 社内共有できる文書・資料に整える | 勉強会資料、活用ガイド |
| 5. 定着化 | 継続利用のルールと更新方法を決める | ベストプラクティス、運用ルール |

## 判断軸

- 実務で使えるか
- 社内メンバーに説明しやすいか
- 安全性、権利、機密情報の扱いに配慮できているか
- 資格試験の理解にもつながるか
