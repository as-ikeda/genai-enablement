## 理解したこと

- 昔のAIは、人間がルールを作り、そのルールに沿って処理するものが中心だった。
- 今のAIは、大量のデータからパターンやルールを学習するものが中心になっている。
- ニューラルネットワークは、人間の脳をまねた計算の仕組みとして理解できる。
- ディープラーニングは、層が深いニューラルネットワークを使った機械学習である。
- 機械学習には、教師あり学習、教師なし学習、半教師あり学習、強化学習がある。
- シナプスのつながりの強さをモデル化したものが「重み」であり、重要な情報を強く扱うことが「情報の重みづけ」と理解できる。
- 過学習は、訓練データに過剰に適合し、新しい未知のデータに対してうまく予測できなくなる現象である。
- 過学習を避ける手法として、アーリーストッピング、正則化、ドロップアウトがある。
- 転移学習は、ある問題で学習したモデルの知識を、別の問題へ活用する手法である。
- 現在普及しているAIの多くは、特定タスクに特化した弱いAI（ANI）である。
- 強いAI（AGI）は、人間のように幅広い知的タスクをこなすAIだが、まだ完全には実現されていない。
- AIの歴史は、ブームと冬を繰り返しながら進展してきた。
- 第三次AIブームは、ビッグデータの活用とディープラーニング技術の発展が大きな要因になっている。

## 用語整理

| 用語 | 自分なりの理解 |
| --- | --- |
| AIとロボットの区別 | AIは判断・推論・学習などの知的処理を担う。ロボットは物理的に動作する機械。AIはロボットに搭載されることもあるが、同じものではない。 |
| ルールベース | 人間があらかじめ決めたルールに従って動く仕組み。 |
| 機械学習 | データからパターンを学び、判断や予測に活用する仕組み。 |
| 教師あり学習 | 正解付きデータから、入力と正解の関係を学習する手法。 |
| 教師なし学習 | 正解ラベルのないデータから、構造や特徴を見つける手法。 |
| クラスタリング | 似たデータ同士をグループ化する教師なし学習の手法。 |
| 次元削減 | 多くの特徴量を、重要な情報を保ちながら少ない軸に整理する手法。 |
| 強化学習 | 行動の結果として得られる報酬をもとに、より良い行動を学ぶ手法。 |
| ニューラルネットワーク | 人工ニューロンをつなげて、入力から出力を計算する仕組み。 |
| 重み | 入力情報をどれくらい重視するかを表す値。 |
| 過学習 | 訓練データに合わせすぎて、未知のデータに弱くなる状態。 |
| アーリーストッピング | 検証データの性能を見ながら、過学習が進む前に学習を止める手法。 |
| 正則化 | モデルを複雑にしすぎないよう制約を加え、過学習を防ぐ手法。 |
| ドロップアウト | 学習中に一部の人工ニューロンをランダムに無効化し、特定の経路に依存しすぎないようにする手法。 |
| 転移学習 | 学習済みモデルを別の問題に応用する手法。 |
| ANI | 特定のタスクに特化した弱いAI。現在の多くのAIが該当する。 |
| AGI | 幅広い知的タスクを人間並みにこなす強いAI。まだ完全には実現されていない。 |
| シンギュラリティ | AIが人間の知能を超え、自己進化し社会に大きな変化をもたらすとされる仮説的な時点。 |

## 試験ポイント

- AIの歴史は問われる可能性があるため、ブーム、時期、代表技術、AIの冬をセットで覚える。
- AI、機械学習、ディープラーニングの関係を説明できるようにする。
- 教師あり学習、教師なし学習、強化学習の違いを整理する。
- 過学習と、その対策であるアーリーストッピング、正則化、ドロップアウトを区別する。
- ANIとAGIの違いを具体例とあわせて覚える。

## 実務での活用

- 現在普及しているAIは、何でも自動で考えてやってくれる存在ではなく、特定タスクに強いANIとして理解する。
- CodexやChatGPTを使う場合も、目的、制約、前提、出力形式を人間が整理して伝える必要がある。
- AI活用では、プロンプトの整理が品質に大きく影響することを再認識した。

## 社内展開ポイント

- 「AI = 何でも自動化してくれるもの」ではなく、「目的を明確に伝えることで力を発揮する補助者」と説明すると伝わりやすい。
- 弱いAI（ANI）と強いAI（AGI）の違いを説明すると、現実的なAI活用の期待値を合わせやすい。
- AI活用ガイドでは、プロンプト設計の重要性を第1章の学習内容と結びつけて説明できそう。
