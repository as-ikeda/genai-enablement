# 生成AI活用推進PJ

生成AIパスポート取得を入口に、実務活用と社内展開までつなげるためのプロジェクト管理リポジトリです。

## 目的

- 生成AIパスポートに合格する
- 実務で生成AIを効果的・安全に使うための知識を整理する
- 社内展開できる勉強会資料・活用ガイド・事例集を作成する

## ディレクトリ

| ディレクトリ | 用途 |
| --- | --- |
| 00_Project | 方針、計画、タスク、週次レビュー |
| 01_Reference | 公式情報、参考資料、リンク集 |
| 02_Study | 学習ログ、試験対策メモ |
| 03_Practice | 実務で試した活用例、検証ログ |
| 04_Output | 社内共有資料、勉強会資料、活用ガイド |
| 05_Exam | 受験計画、模試結果、弱点管理 |
| 06_PromptLibrary | よく使うプロンプト、改善例 |
| 07_BestPractice | 安全利用、レビュー観点、社内向けベストプラクティス |

## 運用

- 学習したら `02_Study` に記録する
- 試験に関係する論点は `05_Exam` に集約する
- 実務で試した内容は `03_Practice` に記録する
- 社内共有できる形に整理した成果物は `04_Output` に置く
- 週1回、`00_Project/WEEKLY_REVIEW.md` を更新して次の行動を決める

## 直近の進め方

1. 公式シラバスと試験範囲を確認する
2. 学習ログを1テーマずつ残す
3. 実務で使えそうな例を小さく試す
4. 社内展開用の資料に転用できる形へ整理する
