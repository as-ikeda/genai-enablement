# プロンプトライブラリ

## 分類

| 分類 | 用途 | 例 |
| --- | --- | --- |
| 調査 | 情報収集、比較、論点整理 | 技術選定、仕様確認 |
| 設計 | 設計案、観点出し、レビュー | 画面設計、DB設計 |
| 実装 | コード作成、リファクタ、テスト | サンプル実装、単体テスト |
| レビュー | コードレビュー、文書レビュー | バグリスク、抜け漏れ確認 |
| 文書作成 | 議事録、手順書、社内資料 | 要約、構成案作成 |
| 学習 | 資格対策、理解確認 | 問題作成、解説 |
