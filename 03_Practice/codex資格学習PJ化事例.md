# Codex活用事例: 資格取得学習計画のPJ化

## 概要

自社で初めてCodexを利用し、生成AIパスポート資格取得に向けた学習計画をプロジェクト化した事例。

単なる学習メモではなく、目的、受験計画、学習スケジュール、教材情報、進捗管理、社内展開資料までをMarkdownで管理し、継続的に更新できる状態にした。

## 背景

- 生成AIパスポートを2026年8月に受験予定
- 学習時間は出社日の移動時間、業後、休日を活用する
- 学習成果を個人の資格取得で終わらせず、社内メンバーがAIを身近に使うためのガイドへ展開したい
- 社内でCodex活用の実例を作りたい

## 利用したAI

| 項目 | 内容 |
| --- | --- |
| ツール | Codex |
| 主な用途 | プロジェクト整理、Markdown作成、学習計画策定、公式情報整理、社内展開観点の整理 |
| 管理形式 | Gitリポジトリ + Markdown |

## 実施したこと

1. プロジェクトフォルダをGitリポジトリとして作成
2. 目的・背景・ゴールをMarkdown化
3. 学習、実務活用、成果物、試験対策のディレクトリを整備
4. 2026年8月受験に向けた学習スケジュールを作成
5. 学習時間の見積もりを実際の生活リズムに合わせて見直し
6. 公式テキストの電子版、GUGA LINEクイズなどスマホ学習手段を整理
7. 社内展開用のAI活用ガイドに転用できる情報として管理

## 得られた効果

| 観点 | 効果 |
| --- | --- |
| 学習計画 | 受験日から逆算した週次計画を作れた |
| 継続性 | 出社移動、業後、休日のどこで何をするか明確になった |
| 情報整理 | 公式情報、教材、学習ログ、弱点管理の置き場ができた |
| 社内展開 | 個人学習をAI活用事例として再利用できる形になった |
| 再現性 | 他資格や別テーマの学習計画にも転用しやすい |

## 社内メンバーに展開できるポイント

- AIに丸投げするのではなく、目的・制約・使える時間を伝えると実行可能な計画に落とせる
- 学習計画、資料整理、進捗管理、振り返りを一つのリポジトリで管理できる
- 公式情報の確認、教材比較、スケジュール作成をAIと一緒に進められる
- 個人の学習成果を、社内ナレッジや勉強会資料へ転用できる

## 注意点

- 試験日程、教材情報、費用などの最新情報は公式サイトで確認する
- AIが作成した計画は、自分の生活リズムに合わせて見直す
- 社内情報、個人情報、顧客情報を不用意に入力しない
- 生成された内容は必ず人が確認してから社内共有する

## 再利用パターン

| 用途 | 例 |
| --- | --- |
| 資格取得 | AWS、Azure、IPA、簿記、業務資格など |
| 技術習得 | 新フレームワーク、新言語、クラウドサービス |
| 社内教育 | 勉強会計画、オンボーディング、ナレッジ整理 |
| 業務改善 | 手順書化、チェックリスト化、FAQ作成 |

## 次に整理すること

- 社内向け説明資料の1枚目に使える要約
- Codexに依頼したプロンプト例
- 他メンバーが真似できる手順テンプレート
- 社内ルールに合わせた入力情報の注意点
